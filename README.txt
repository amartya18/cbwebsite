How to Setup Development Setup Environment:

requirement:
- Node.js and npm
to check, run: 
node -v
this will return ur current version of node

run these commands:
sudo npm install gulp --global
npm install 

* gulp and browser-sync are for development only

* to run dev server type 'gulp' in the terminal and open localhost:6969

then ur good to go; the dev server will auto reload every html or css changes :D
