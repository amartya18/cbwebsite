How to Setup Development Setup Environment:

requirement:
- Node.js and npm
to check run: 
node -v
this will return ur current version of node

run this command:
sudo npm install gulp --global
npm install 

* gulp browser-sync gulp-sass are for development only

* to run dev server type 'gulp' in the terminal and open localhost:3010

then ur good to go; the dev server will auto reload every html or sass changes and yes u can use SASS :D
